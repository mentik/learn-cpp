## Learning C++

Let's learn C++

## Requirements

- Linux/WSL
- g++ >= 14.2.0 [https://gcc.gnu.org/]

    C++ compiler, support C++ 20.

    ```sh
    g++ --version
    ```
- GDB: The GNU Project Debugger [https://www.sourceware.org/gdb/]

    For debugging.

- VSCode

    This setup run the CPP file as a project.

    Simply open the a CPP file, run, the result will be in `build` directory.

## Manual Run Example

### Compile

```sh
g++ src/hello-world/hello-world.cpp -o build/main
```

### Run 

```sh
./build/main
```
